# Guide to ComfyUI - Grid

In this tutorial, we will show how to create a 2D parameter grid with ComfyUI and a Python script. Since ComfyUI is more node-oriented, the interface is not friendly to loops. However, we can use the workflow in API format to make requests using a Python script. 

## API JSON file

All API JSON files are contained in the folder `workflows-api` of this repository. For example, below we have the beginning of the file https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/workflows-api/txt2img_canon.json:

```json
{
  "3": {
    "inputs": {
      "seed": 566510339945522,
      "steps": 20,
      "cfg": 5,
      "sampler_name": "euler",
      "scheduler": "normal",
      "denoise": 1,
      "model": [
        "11",
        0
      ],
      "positive": [
        "6",
        0
      ],
      "negative": [
        "7",
        0
      ],
      "latent_image": [
        "5",
        0
      ]
    },
    "class_type": "KSampler",
    "_meta": {
      "title": "KSampler"
    }
  },
```

As mentioned in many other tutorials, you can use the script [run_workflow.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/run_workflow.py) to run any API JSON workflows with the command `python run_workflow.py "{path_to_workflow_json}"`. However, this is not efficient to produce grids. 

## Text to Image Grid

First, you need to put the file [grid_txt2img.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_txt2img.py) in the same location as the executable (e.g. `run_nvidia_gpu.bat`). The script content starts as follows:

```python
### Constants ###
SERVER_URL = "http://127.0.0.1:8188"
API_JSON_FILE = "ComfyUI/user/default/workflows-api/txt2img_canon.json"
BIG_GRID_FILE = f"ComfyUI/output/grid_{datetime.datetime.now().strftime('%Y-%m-%d-%H_%M_%S')}.png"


### Parameters ###

# Grid parameters.
param1 = {
    "3": {
        "inputs": {
            "steps": [10, 20, 30, 50]
        }
    }
}
param2 = {
    "3": {
        "inputs": {
            "cfg": [2, 7, 12, 20]
        }
    }
}

# Optional parameters.
param_opt = {
    "6": {    
        "inputs": {
            "text":
                "best quality, amazing quality, perfect eyes, blushing, foreground, vibrant color palette, soft shadows and subtle highlights, light depth, solo, looking at viewer, cowboy shot, dynamic pose, girl, long hair, pink hair, purple hair, gradiant hair, twintails, kneeling on bed, bedroom, smile"
        }
    },
    "7": {    
        "inputs": {
            "text":
                "noisy, blurry, soft, deformed, ugly, low detailed, bad quality, watermark, long neck, multiple legs, multiple fingers, missing limbs, deformed hands, deformed feet, mutations ugly hands, multiple navels"
        }
    }
}
```

Usually you only need to change the variables `param1`, `param2`. Optionally, you can change `param_opt`, which will fix other parameters during the grid iterations. Note that the nested key structure must match the API JSON file. Once you have finished editing, save the file and run the command `python grid_txt2img.py`.

## Image to Image Grid

This one is very similar to the previous method, with the only difference that now you need to use the file [grid_img2img.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_img2img.py) and the corresponding API JSON file `img2img_canon.json` for the workflow. Also, you need to provide the path to the input image. Once you have finished editing, save the file and run the command `python grid_img2img.py`. 

Below we have an example of the grid generated by this script. The following parameters were used to generate this grid:

```python
# Grid parameters.
param1 = {
    "6": {
        "inputs": {
            "text": [
                "goth girls, park in the background, sunny day, masterpiece, high quality, pixelart, detailed",
                "goth monsters, park in the background, dark day, masterpiece, high quality, pixelart, detailed",
                "monster, park in the background, dark day, masterpiece, high quality, pixelart, detailed"
            ]
        }
    }
}
param2 = {
    "3": {
        "inputs": {
            "cfg": [2, 7, 12, 20]
        }
    }
}

# Input parameters.
image_path = "...picture.jpg"
```

## Text to Video Grid

Just like the others, you only need to use the file [grid_txt2vid.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_txt2vid.py) and the corresponding API JSON file `txt2vid_canon.json` for the workflow. 

Besides `param1`, `param2` and `param_opt` you may want to change the value of `LOOPS`. This value determines how many times the generated video will repeat in the file. Once you have finished editing, save the file and run the command `python grid_txt2vid.py`. 

Below we have an example of the grid generated by this script. 

https://github.com/user-attachments/assets/fe12b1ea-3d8e-4b98-bffc-481e1f52673f

> PS: Since this script concatenates and plays several videos at the same time, you cannot change the resolution or the FPS in the grid.

## FFLF Grid

Just like the others, you only need to use the file [grid_FFLF.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_FFLF.py) and the corresponding API JSON file `FFLF.json` for the workflow. The same observations made for Image To Video Grid still applies here, with the difference that now you need to provide two image files and their corresponding keys in the JSON workflow.

```python

### Constants ###
SERVER_URL = "http://127.0.0.1:8188"
API_JSON_FILE = "ComfyUI/user/default/workflows-api/FFLF.json"
BIG_GRID_FILE = f"ComfyUI/output/grid_{datetime.datetime.now().strftime('%Y-%m-%d-%H_%M_%S')}.mp4"
TEMP_VIDEO_DIR = "ComfyUI/output"


### Parameters ###

# Grid parameters.
param1 = {
    "86": {
        "inputs": {
            "cfg": [1]
        }
    }
}
param2 = {
    "86": {
        "inputs": {
            "noise_seed": [165697863152, 65475646262, 8978656434536]
        }
    }
}

# Input parameters.
image_path1 = "...img1.png"
image_path2 = "...img2.png"

param_input = {
    "97": {    # first frame
        "inputs": {
            "image": image_path1
        }
    },
    "142": {   # last frame
        "inputs": {
            "image": image_path2
        }
    }
}
```

Once you have finished editing, save the file and run the command `python grid_FFLF.py`.

## Inpainting Grid

To perform inpainting grid, it is recommended to first create the mask with the default workflow for inpainting as explained [here](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/Guide-ComfyUI-Inpainting.md). After you finish editing the masked image, it will be save on the path `ComfyUI\input\clipspace`. The example below illustrates how to configure the script.

```python
# Grid parameters.
param2 = {
    "3": {
        "inputs": {
            "cfg": [0.5, 1, 2, 4, 8, 16, 32, 64]
        }
    }
}
param1 = {
    "3": {
        "inputs": {
            "denoise": [0.4, 0.6, 0.8]
        }
    }
}

# Input parameters.
image_path = "...\\ComfyUI\\input\\clipspace\\clipspace-painted-masked-1783168940260.png"
```

Once you have finished editing, save the file and run the command `python grid_inpainting.py`.

## IP-Adapter Grid

You only need to use the file [grid_FFLF.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_IPAdapter.py) and the corresponding API JSON file `IPAdapter.json` for the workflow. The same observations made for Image To Video Grid still applies here, with the difference that now you need to provide two image files and their corresponding keys in the JSON workflow.

```python

### Constants ###
SERVER_URL = "http://127.0.0.1:8188"
API_JSON_FILE = "ComfyUI/user/default/workflows-api/IPAdapter.json"
BIG_GRID_FILE = f"ComfyUI/output/grid_{datetime.datetime.now().strftime('%Y-%m-%d-%H_%M_%S')}.png"


### Parameters ###

# Grid parameters. 
param1 = {
    "22": {
        "inputs": {
            "weight": [0.1, 0.5, 0.9]
        }
    }
}
param2 = {
    "22": {    
        "inputs": {
            "preset": [
                "STANDARD (medium strength)",
                "PLUS (high strength)",
                "PLUS FACE (portraits)",
                "COMPOSITION",
                "FACEID"
            ]
        }
    }
}

# Optional parameters.
param_opt = {
    "22": {
        "inputs": {
            "start_at": 0.3
        }
    },
    "22": {
        "inputs": {
            "end_at": 0.8
        }
    },
    "23": {
        "inputs": {
            "image": "...picture.png"
        }
    },
}
```

Once you have finished editing, save the file and run the command `python grid_IPAdapter.py`.
