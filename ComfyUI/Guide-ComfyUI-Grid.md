# Guide to ComfyUI - Grid

In this tutorial, we will show how to create a 2D parameter grid with ComfyUI and a Python script. Since ComfyUI is more node-oriented, the interface is not friendly to loops. However, we can use the workflow in API format to make requests using a Python script. 

## API JSON file

All API JSON files are contained in the folder `workflows-api` of this repository. For example, below we have the beginning of the file https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/workflows-api/txt2img_canon.json:

```
{
  "3": {
    "inputs": {
      "seed": 566510339945522,
      "steps": 20,
      "cfg": 5,
      "sampler_name": "euler",
      "scheduler": "normal",
      "denoise": 1,
      "model": [
        "11",
        0
      ],
      "positive": [
        "6",
        0
      ],
      "negative": [
        "7",
        0
      ],
      "latent_image": [
        "5",
        0
      ]
    },
    "class_type": "KSampler",
    "_meta": {
      "title": "KSampler"
    }
  },
```

As mentioned in many other tutorials, you can use the script [run_workflow.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/run_workflow.py) to run any API JSON workflows with the command `python run_workflow.py "{path_to_workflow_json}"`. However, this is not efficient to produce grids. 

## Text to Image Grid

First, you need to put the file [grid_txt2img.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_txt2img.py) in the same location as the executable (e.g. `run_nvidia_gpu.bat`). The script content starts as follows:

```python
# Server address.
SERVER_URL = "http://127.0.0.1:8188"

param1 = {"3": {"inputs": {"steps": [10, 20, 30, 50]}}}
param2 = {"3": {"inputs": {"cfg": [2, 7, 12, 20]}}}
image_path = "...\\picture.jpg"
api_json_file = "ComfyUI/user/default/workflows-api/txt2img_canon.json"
big_grid_file = f"ComfyUI/output/grid_{datetime.datetime.now().strftime('%Y-%m-%d-%H_%M_%S')}.png"
```

Usually you only need to change the variables `param1` and `param2` to the parameters you want to change. Note that the nested key structure must match the API JSON file. Once you have finished editing, save the file and run the command `python grid_txt2img.py`.

## Image to Image Grid

This one is very similar to the previous method, with the only difference that now you need to use the file [grid_img2img.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_img2img.py) and the corresponding API JSON file `img2img_canon.json` for the workflow. Once you have finished editing, save the file and run the command `python grid_img2img.py`. 

Below we have an example of the grid generated by this script. The following parameters were used to generate this grid:

```
param1 = {"6": {"inputs": {"text": 
    ["goth girls, park in the background, sunny day, masterpiece, high quality, pixelart, detailed",
     "goth monsters, park in the background, dark day, masterpiece, high quality, pixelart, detailed",
     "monster, park in the background, dark day, masterpiece, high quality, pixelart, detailed"]
    }}}
param2 = {"3": {"inputs": {"cfg": [2, 7, 12, 20]}}}
```

## Text to Video Grid

Just like the others, you only need to use the file [grid_txt2vid.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_txt2vid.py) and the corresponding API JSON file `txt2vid_canon.json` for the workflow. The script content starts as follows:

```python
# Server address.
SERVER_URL = "http://127.0.0.1:8188"

# Parameters to vary.
# param1 = rows, param2 = columns.
param1 = {"81": {"inputs": {"cfg": [1, 3, 6]}}}
param2 = {"86": {"inputs": {"shift": [1, 2, 5]}}}
image_path = "...\\picture.jpg"
api_json_file = "ComfyUI/user/default/workflows-api/txt2vid_canon.json"
big_grid_file = f"ComfyUI/output/grid_{datetime.datetime.now().strftime('%Y-%m-%d-%H_%M_%S')}.mp4"

temp_video_dir = "ComfyUI/output"

# Video grid options.
LOOPS = 5
```

Besides `param1` and `param2` you may want to change the value of `LOOPS`. This value determines how many times the generated video will repeat in the file. Once you have finished editing, save the file and run the command `python grid_txt2vid.py`. 

Below we have an example of the grid generated by this script. 

https://github.com/user-attachments/assets/fe12b1ea-3d8e-4b98-bffc-481e1f52673f

> PS: Since this script concatenates and plays several videos at the same time, you cannot change the resolution or the FPS in the grid.

## Image to Video Grid

Finally the last type of grid generation! Just like the others, you only need to use the file [grid_img2vid.py](https://github.com/felipebottega/AI-Audiovisual-Lab/blob/main/ComfyUI/scripts/grid_img2vid.py) and the corresponding API JSON file `img2vid_canon.json` for the workflow. The same observations made for Text To Video Grid still applies here. Additionally, it is necessary to change the path of the image file you intend to use (in the API JSON file), and you must use the absolute path to the file. Once you have finished editing, save the file and run the command `python grid_img2vid.py`.
